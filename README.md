# Customer-Segmentation-ML
customer segmentation using K-means and Hierarchical Clustering
this project applies unsupervised machine learning techniques to segment customers based on :
-Annual income 
-Spending score 

## Algorithms Used 
- K-means Clustering
- -Hierarchical Clustering

  ## Objective
  To understand customer behaviour and identify meaningful segments for business decision making .

  ## Tools
  Python,Pandas,Numpy,Matplotlib,scikit-learn
